El SDK oficial de pan para JavaScript/TypeScript simplifica la integración con pan API.

## Instalacion

<CodeGroup>
```bash npm
npm install @odisea-labs/pan
```

```bash yarn
yarn add @odisea-labs/pan
```

```bash pnpm
pnpm add @odisea-labs/pan
```

```bash bun
bun add @odisea-labs/pan
```
</CodeGroup>

## Configuración Basica

```typescript
import { Pan } from '@odisea-labs/pan';

const pan = new Pan({
  apiKey: process.env.PAN_API_KEY
});
```

## Opciones de Configuración

```typescript
const pan = new Pan({
  // Requerido
  apiKey: 'pan_sk_...',

  // Opcionales
  baseURL: 'https://api.pan.com',  // URL base de la API
  debug: false,                     // Modo debug (default: false)
  network: 'mainnet',              // Red: 'mainnet' | 'testnet' (default: 'mainnet')
});
```

| Opcion | Tipo | Default | Descripción |
|--------|------|---------|-------------|
| `apiKey` | string | - | Tu API key de pan |
| `baseURL` | string | `https://api.pan.com` | URL base |
| `debug` | boolean | false | Activa logs de debug |
| `network` | `"mainnet"` \| `"testnet"` | `"mainnet"` | Red a utilizar |

## Verificar Instalación

```typescript
import { Pan } from '@odisea-labs/pan';

const pan = new Pan({ apiKey: process.env.PAN_API_KEY });

// Verificar conexión
try {
  const yields = await pan.yields.getAll();
  console.log('Conexión exitosa!');
  console.log('Mejor APY:', yields.best.apy);
} catch (error) {
  console.error('Error de conexión:', error.message);
}
```

## Variables de Entorno

Recomendamos usar variables de entorno para la API key:

```env
# .env
PAN_API_KEY=pan_sk_tu_api_key_aqui
```

```typescript
// Cargar con dotenv o similar
import 'dotenv/config';
import { Pan } from '@odisea-labs/pan';

const pan = new Pan({
  apiKey: process.env.PAN_API_KEY
});
```

<Warning>
  Nunca commitees tu API key. Agrega `.env` a tu `.gitignore`.
</Warning>

## TypeScript

El SDK incluye tipos TypeScript completos:

```typescript
import { Pan } from '@odisea-labs/pan';
import type { Wallet, Intent, WalletBalances, AllYieldsResponse } from '@odisea-labs/pan';

const pan = new Pan({ apiKey: process.env.PAN_API_KEY! });

// Los tipos se infieren automaticamente
const wallet: Wallet = await pan.wallet.create({
  userId: 'user_123'
});

const balances: WalletBalances = await pan.wallet.getBalances(wallet.id);

const intent = await pan.lend({
  walletId: wallet.id,
  amount: 1000
});
```
