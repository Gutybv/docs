## Descripción Planificada

Fondea una wallet con tokens de testnet para pruebas.

<Info>
  Cuando esté disponible, solo funcionará en testnet. No estará disponible en produccion.
</Info>

## Endpoint Planificado

```
POST https://api.pan.com/api/v1/demo/fund
```

## Request Planificado

```json
{
  "walletId": "pan_wallet_abc123"
}
```

### Parámetros

| Campo | Tipo | Requerido | Descripción |
|-------|------|-----------|-------------|
| `walletId` | string | Si | ID de la wallet pan |

## Response Planificado

### 200 OK

```json
{
  "success": true,
  "walletId": "pan_wallet_abc123",
  "amount": 1,
  "chain": "arbitrum-sepolia",
  "asset": "USDC",
  "txHash": "0x123abc...",
  "explorerUrl": "https://sepolia.arbiscan.io/tx/0x123abc..."
}
```

## Alternativa Actual

Mientras este endpoint no esté disponible, puedes fondear wallets de testnet usando faucets externos:

1. Obtener la dirección de tu wallet con `GET /api/v1/wallets/:userId`
2. Usar un faucet de testnet para enviar tokens a esa dirección
3. Verificar el balance con `GET /api/v1/balances/:walletId`
