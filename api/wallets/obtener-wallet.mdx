---
title: "Obtener Wallet"
description: "GET /api/v1/wallets/:userId"
---

Obtiene la wallet asociada a un usuario.

## Endpoint

```
GET https://api.pan.com/api/v1/wallets/:userId
```

## Autenticación

API Key o X402

## Parámetros de URL

| Parámetro | Tipo | Descripción |
|-----------|------|-------------|
| `userId` | string | ID del usuario en tu app |

## Response

### 200 OK

```json
{
  "id": "pan_wallet_a1b2c3d4e5f6",
  "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
  "chainType": "ethereum",
  "chains": [
    "ethereum",
    "ethereum-sepolia",
    "arbitrum",
    "arbitrum-sepolia",
    "base",
    "base-sepolia"
  ],
  "createdAt": "2024-01-15T10:30:00Z"
}
```

### Campos de Respuesta

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `id` | string | ID único de la wallet |
| `address` | string | Dirección de la wallet |
| `chainType` | string | Tipo de chain (`ethereum` o `tron`) |
| `chains` | string[] | Chains soportadas por esta wallet |
| `createdAt` | string | Fecha de creación |

### Errores

| Código | Error | Descripción |
|--------|-------|-------------|
| 401 | UNAUTHORIZED | API key inválida |
| 404 | NOT_FOUND | Usuario no tiene wallet |

## Ejemplos

<CodeGroup>
```bash cURL
curl -X GET "https://api.pan.com/api/v1/wallets/usuario_123" \
  -H "Authorization: Bearer $PAN_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.pan.com/api/v1/wallets/usuario_123',
  {
    headers: {
      'Authorization': `Bearer ${process.env.PAN_API_KEY}`
    }
  }
);

const wallet = await response.json();
```

```typescript SDK
import Pan from '@odisea-labs/pan';

const pan = new Pan({ apiKey: process.env.PAN_API_KEY });

const wallet = await pan.wallet.get('usuario_123');
```
</CodeGroup>
