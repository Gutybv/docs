title: "Obtener Yields"
description: "GET /v1/yields"
---

Obtiene las tasas APY disponibles en protocolos DeFi.

## Endpoint

```
GET https://api.pan.dev/v1/yields
```

## Autenticacion

API Key o X402

### Campos de Yield

| Campo | Tipo | Descripcion |
|-------|------|-------------|
| `chain` | string | Nombre de la chain |
| `chainId` | number | ID numerico de la chain |
| `protocol` | string | Protocolo DeFi |
| `protocolVersion` | string | Version del protocolo |
| `asset` | string | Token soportado |
| `assetAddress` | string | Direccion del token |
| `apy` | number | APY total (base + rewards) |
| `apyBase` | number | APY base del protocolo |
| `apyReward` | number | APY de recompensas |
| `tvl` | number | Total Value Locked en USD |
| `riskLevel` | string | `low`, `medium`, o `high` |

## Ejemplos

<CodeGroup>
```bash cURL
curl -X GET "https://api.pan.dev/v1/yields" \
  -H "Authorization: Bearer $PAN_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.pan.dev/v1/yields', {
  headers: {
    'Authorization': `Bearer ${process.env.PAN_API_KEY}`
  }
});

const { yields, bestYield } = await response.json();
console.log('Mejor APY:', bestYield.apy, '% en', bestYield.chain);
```

```typescript SDK
const yields = await pan.yields.getAll();
console.log('Mejor APY:', yields.bestYield.apy);
```
</CodeGroup>

## Notas

- Los APYs se actualizan cada 5 minutos
- `bestYield` indica la mejor oportunidad actual
- Cache recomendado: 1-5 minutos
